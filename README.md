# Smart Expense Tracker

Smart Expense Tracker is an Android application designed to help users manage daily expenses efficiently through simple expense logging, category organization, and intuitive financial tracking.

## Overview
This project was built to provide an easy-to-use personal finance management solution for tracking spending habits, organizing expenses, and improving budgeting awareness.

## Features
- Add and save daily expenses
- Categorize expenses for better tracking
- Clean and user-friendly Android interface
- Organized expense management
- Practical personal finance utility

## Tech Stack
- Android Studio
- Java 
- XML (UI Design)
- SQLite
